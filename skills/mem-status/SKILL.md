# Memory Status

메모리 시스템의 현재 상태를 확인합니다.

## 사용법

```
/mem-status
/mem-status --detailed
```

## 출력 정보

- 총 세션 수
- 총 관찰 수
- 타입별 관찰 분포
- 스토리지 사용량
- 최근 활동

## 옵션

- `--detailed` - 상세 통계 표시
- `--json` - JSON 형식으로 출력
